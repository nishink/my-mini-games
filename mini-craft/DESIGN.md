# mini-craft - デザインメモ

自由なブロック配置を楽しむサンドボックス・ミニゲーム。限られたエリア内で自分だけの造形を作成する。

## 概要
- ジャンル: 2.5Dアイソメトリック・サンドボックス
- 目的: エリア内にブロックを配置して好きな形を作る。
- 操作: 
    - ブロック配置: マウス左クリック / タップ
    - ブロック削除: Shift+クリック / 長押し
    - 種類選択: 数字キー / 画面上のインベントリ・アイコン
    - カメラ移動: 矢印キー（任意）

## 実装要素
1. アイソメトリック描画エンジン（2D座標を2.5D投影）
2. ブロックの積層システム（高さの概念）
3. インベントリ（複数のブロックタイプ：土、草、レンガ、水、木など）
4. マウス座標からグリッド座標への変換（逆投影）
5. 簡易保存（LocalStorage）

## クラス構成
- `Game`: 全体の統括、入力イベント管理
- `World`: ブロックデータの保持（3次元配列 `[x][y][z]`）
- `Renderer`: アイソメトリック描画の実行
- `UI`: インベントリ表示、ブロック選択の管理

## ブロックデータ構造
```javascript
// [x][y][z] = typeIndex (0: empty, 1: grass, 2: stone...)
this.data[x][y][z] = 1; 
```

## 開発フェーズ
1. プロジェクト構造作成
2. `World` クラスの実装（3D配列によるデータ管理）
3. アイソメトリック描画の基本（平面グリッドの表示）
4. 積み上げ描画（高さ $Z$ に応じた描画位置のオフセット）
5. 入力処理（マウスクリックした場所への設置・削除）
6. インベントリUIとブロック種類の切り替え
7. 簡易セーブ機能の実装
8. モバイル対応（タップ操作の調整）
