# ミニジャンパー (mini-jumper) 設計企画書

## 🎮 コンセプト
- 常に上に跳ね続けるキャラクターを操作し、足場を乗り継いでどこまで高く登れるかを競うアクションゲーム。
- 画面の下端に落ちるとゲームオーバー。

## ✍️ 操作
- 矢印キー (←→) または A/Dキー: 左右移動
- 画面端を通り抜けると反対側から出てくる（ラップアラウンド）

## 🧩 ゲーム仕様
1. **ジャンプ物理**
   - キャラクターには常に下向きの重力がかかる。
   - 足場（プラットフォーム）を上から踏むと、上向きの力が加わりジャンプする。
2. **スクロール（カメラ）**
   - キャラクターが画面の半分より上に到達すると、世界が下にスクロールする。
3. **プラットフォーム生成**
   - スクロールに合わせて、上方に新しい足場をランダムに生成。
   - 高くなるにつれて、足場の幅が狭くなったり、動く足場が登場する。
4. **スコア**
   - 到達した最高高度がスコアとなる。
