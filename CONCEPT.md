# Mini Games Collection - コンセプトメモ

このプロジェクトでは、1〜3日程度でコア部分を完成させられる「小さくてすぐに遊べるゲーム」を多数開発していきます。
既存のロジックを再利用し、効率的にバリエーションを増やしていくことを目指します。

## 🎮 開発済みタイトル
1. **mini-action** (横スクロールアクション)
   - ステータス: 完成 (v0.3)
   - 特徴: ジャンプ、ショット、ランダム地形生成、レベル制。
2. **mini-roguelike** (ターン制ローグライク)
   - ステータス: 完成 (基盤実装済み)
   - 特徴: グリッド移動、ターン管理、マップ生成、アイテム/装備システム。
3. **mini-snake** (古典的アクション)
   - ステータス: 完成 (v0.1)
   - 特徴: ヘビの成長、グリッド移動、ハイスコア機能。

## 🚀 今後の開発リスト (予定)
1. **mini-shmup** (縦スクロールシューティング)
   - 迫り来る敵を撃ち落とす。
   - 技術要素: 弾幕生成、当たり判定（mini-actionの応用）。
3. **mini-sokoban** (倉庫番パズル)
   - 荷物を指定位置に運ぶ。
   - 技術要素: グリッド移動ロジック（mini-roguelikeの応用）。
4. **mini-clicker** (放置・クリッカー)
   - クリックでポイントを貯めて施設を買う。
   - 技術要素: UI/数値計算、インフレ演出。
5. **mini-defense** (タワーディフェンス)
   - 攻めてくる敵を砲台で迎え撃つ。
   - 技術要素: 敵の経路探索、自動攻撃ロジック。
