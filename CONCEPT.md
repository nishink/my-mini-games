# Mini Games Collection - コンセプトメモ

このプロジェクトでは、1〜3日程度でコア部分を完成させられる「小さくてすぐに遊べるゲーム」を多数開発していきます。
既存のロジックを再利用し、効率的にバリエーションを増やしていくことを目指します。

## 🎮 開発済みタイトル
1. **mini-action** (横スクロールアクション)
   - ステータス: 完成 (v0.3)
   - 特徴: ジャンプ、ショット、ランダム地形生成、レベル制。
2. **mini-roguelike** (ターン制ローグライク)
   - ステータス: 完成 (基盤実装済み)
   - 特徴: グリッド移動、ターン管理、マップ生成、アイテム/装備システム。
3. **mini-snake** (古典的アクション)
   - ステータス: 完成 (v0.1)
   - 特徴: ヘビの成長、グリッド移動、ハイスコア機能。
4. **mini-shmup** (縦スクロールシューティング)
   - ステータス: 完成 (v0.1)
   - 特徴: 縦スクロール、敵弾回避、連射ショット、パワーアップ要素（基礎）。
5. **mini-sokoban** (倉庫番パズル)
   - ステータス: 完成 (v0.1)
   - 特徴: クラシックな倉庫番、ステージ制、リセット機能。
6. **mini-clicker** (放置・クリッカー)
   - ステータス: 完成 (v0.1)
   - 特徴: 数値のインフレ、施設購入、自動セーブ機能。
