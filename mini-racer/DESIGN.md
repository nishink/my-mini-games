# ミニレーサー (mini-racer) 設計企画書

## 🎮 コンセプト
- 見下ろし視点（トップダウン）のカーレースゲーム。
- 慣性のある操作感を楽しみながら、障害物を避けてハイスコアを目指す。

## ✍️ 操作
- 矢印キー (↑): アクセル（前進）
- 矢印キー (↓): ブレーキ / バック
- 矢印キー (←→): ステアリング（旋回）

## 🧩 ゲーム仕様
1. **車両物理**
   - 加速度、最高速度、摩擦（減速）の概念。
   - 旋回は現在の速度に応じて曲がり具合が変わる（高速ほど大回り）。
2. **コース**
   - 縦に無限に続くコース。左右に壁がある。
   - 速度が上がるにつれて背景が速く流れるスクロール演出。
3. **障害物**
   - 他の車やオイル、壁などの障害物。
   - 衝突すると減速またはゲームオーバー。
4. **スコア**
   - 走行距離がスコアとなる。
