# ミニ倉庫番パズル (mini-sokoban) 設計企画書

## 🎮 コンセプト
- クラシックな倉庫番パズルゲーム。
- プレイヤーはすべての荷物を指定されたゴール（目的地）に運ぶ。
- 荷物は「押す」ことしかできず、「引く」ことはできない。
- 一度に押せる荷物は1つだけ。

## ✍️ 操作
- 矢印キー (↑↓←→) または WASD: 移動 / 荷物を押す
- Rキー: ステージのリセット（詰んだ場合）
- Zキー: 1手戻る（Undo） ※余裕があれば実装

## 🧩 ゲーム仕様
1. ステージはタイルマップ（壁、床、目的地、荷物、プレイヤー）。
2. プレイヤーが荷物の方向に移動しようとしたとき、その先に障害物がなければ荷物を押すことができる。
3. すべての目的地に荷物が置かれた時点でステージクリア。
4. ステージ制で、クリアするごとに難易度が上がる。

## 🛠 技術的アプローチ
- HTML5 Canvas または DOMタイル 描画（今回はCanvasを採用）。
- JavaScript ES modules (`Game`, `LevelManager`, `Entity`, `Renderer`, `Input`)。
- `mini-roguelike` のグリッド座標管理と `Entity` クラスを応用。
