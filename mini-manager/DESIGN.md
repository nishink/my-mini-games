# ミニマネージャー (mini-manager) - ちいさな錬金工房 設計企画書

## 🎮 コンセプト
- ファンタジー世界のポーション屋「ちいさな錬金工房」の店主となり、冒険者たちに薬を売って利益を上げる経営シミュレーション。
- 種類の異なるポーションをどう仕入れ、限られた棚のスペースをどう活用するかが鍵。

## ✍️ 操作
- 画面上のボタンをクリックして「仕入れ」「設備投資」「店を開ける」などのアクションを実行。

## 🧩 ゲーム仕様
1. **リソース管理**
   - **資金 (Gold)**: ポーションの仕入れやお店の強化に使用。
   - **棚の容量 (Shelf Space)**: すべての商品で共有される最大在庫数。
   - **在庫 (Stock)**: 「赤のポーション(HP)」と「青のポーション(MP)」の2種類。
   - **評判 (Reputation)**: お客さん（冒険者）の来店数に直接影響。
2. **商品ラインナップ**
   - **赤のポーション**: 仕入れが安く(10G)、多くの冒険者が買い求める。薄利多売。
   - **青のポーション**: 仕入れが高く(40G)、特定の客（魔導士）のみが買うが、利益が大きい。
3. **1日の流れ**
   - **準備フェーズ**: 資金を使って仕入れや広告（評判UP）、棚の拡張を行う。
   - **営業フェーズ**: その日の来客数、売上、発生したイベントの結果を確認。
4. **ランダムイベント**
   - 「ギルドの遠征（赤の需要増）」「魔法学会（青の需要増）」「嵐（客足減）」など、ファンタジー世界ならではの出来事が発生。
