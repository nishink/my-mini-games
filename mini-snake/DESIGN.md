# ミニスネークゲーム設計企画書

## 🎮 コンセプト
- 古典的なスネークゲーム。
- ヘビを操作してエサ（リンゴ）を食べ、体を長くしていく。
- 壁や自分の体に当たるとゲームオーバー。

## ✍️ 操作
- 矢印キー (↑↓←→) または WASD: 移動方向の変更。

## 📋 実装状況チェックリスト
- [x] ヘビの移動 (1マスずつ進む)
- [x] 方向転換 (逆方向への反転防止あり)
- [x] リンゴ生成と体の延長
- [x] 壁との衝突判定
- [x] 自分の体との衝突判定
- [x] スコア加算

- [ ] 速度上昇 (❌ 未実装)
