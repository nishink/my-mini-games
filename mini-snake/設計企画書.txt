# ミニスネークゲーム設計企画書

## 🎮 コンセプト
- 古典的なスネークゲーム。
- ヘビを操作してエサ（リンゴ）を食べ、体を長くしていく。
- 壁や自分の体に当たるとゲームオーバー。

## ✍️ 操作
- 矢印キー (↑↓←→) または WASD: 移動方向の変更。

## 🧩 ゲーム仕様
1. グリッドベースの移動（例: 20x20マス）。
2. エサを食べるとスコアが加算され、体が1節分伸びる。
3. エサを食べると新しいエサがランダムな空きマスに再配置される。
4. 移動速度は一定（またはスコアに応じて微増）。
5. ハイスコアを記録。

## 🛠 技術的アプローチ
- HTML5 Canvas を使用した描画。
- JavaScript ES modules による構造化。
- `mini-roguelike` のグリッド移動ロジックを簡略化して流用。
